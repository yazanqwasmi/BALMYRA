## Overview

Official website for BALMYRA — a responsive, multi-section brand site built with a clean frontend and PHP backend. Includes automated deployment via GitHub Actions CI/CD on every push to `main`.

## Tech Stack

**Frontend:** HTML5, CSS3, JavaScript  
**Backend:** PHP 7.4+  
**CI/CD:** GitHub Actions  
**Assets:** Optimized images and static resources

## Project Structure

```
├── index.html          # Main entry point
├── assets/             # Images, fonts, and static files
│   └── ...
├── .github/
│   └── workflows/      # CI/CD pipeline configuration
```

## Local Development

```bash
git clone https://github.com/yazanqwasmi/BALMYRA.git
cd BALMYRA

# Run local dev server
php -S localhost:8000
```

Open `http://localhost:8000` in your browser.

## Deployment

Automated via GitHub Actions — every push to `main` triggers a deployment pipeline.
